# Contributing to CeloHT Education

## Contribution Rules
- All content must align with Celo, cUSD, and Valora
- No speculative language
- No external chains unless for comparison only

## Workflow
1. Fork the repository
2. Create a branch: `feature/<name>` or `fix/<name>`
3. Add or improve content
4. Submit a Pull Request

## Review Criteria
- Educational clarity
- Ecosystem alignment
- Cultural relevance
