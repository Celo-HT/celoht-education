# Contributing to CeloHT Education

## Contribution Rules
- All content **must strictly align** with the Celo ecosystem: **Celo, cUSD, and Valora**.
- No speculative, promotional, or financial-advice language.
- External blockchains may be mentioned **for comparison only**, never as alternatives.
- Content must be educational, factual, and verifiable.
- No token creation, trading signals, or investment promises.

## Workflow
1. Fork the repository.
2. Create a branch using one of the following formats:
   - `feature/<short-description>`
   - `fix/<short-description>`
3. Add, update, or improve educational content.
4. Ensure clarity, accuracy, and consistency.
5. Submit a Pull Request with a clear description of changes.

## Review Criteria
- Educational clarity and correctness.
- Alignment with the Celo ecosystem principles.
- Cultural relevance for the Haitian and emerging-market context.
- Clear language and structured formatting.
